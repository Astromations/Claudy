# Changelog

All notable changes to the Claudy theme will be documented here.

---

## [1.0.0] — 2026-03-25

### Initial release

- Full editor chrome: sidebar, activity bar, tabs, title bar, status bar, panels
- Complete terminal ANSI palette matched to theme colours
- Token colours for all major languages: JS/TS/JSX/TSX, Python, Rust, Go, HTML, CSS, JSON, Markdown, Diff
- Semantic highlighting enabled with full token type coverage
- Git decoration colours
- Minimap, breadcrumbs, peek view, suggest widget, hover widget all themed
- Input fields, dropdowns, buttons, scrollbars, and badges themed consistently
